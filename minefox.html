Minefox Network

The Minefox Network aims to be an ambitious, original Minecraft Server network full of features and amazing minigames that hundreds of players can play at the same time, for fun. The Server network includes many minigames as well as additional features such as Cosmetics and other Services. The server contains a huge variety of minigames, some of them including PVP, but the main Miniame, named Realms is a really exciting one, where players can compete against each other and Raid each other's Islands (named Guilds), featuring Towers, enchants, sockets, event worlds, resource worlds, mobs, special items, store, legendary weapons, 3D textures and much much more. Other ames are available too such as Minefox Royale, or Kingdoms. The server is currently offline, and no re release date is available as of September 25, 2019.
